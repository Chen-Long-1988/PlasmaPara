# 等离子体参数计算器

一个简单的离线 HTML 工具，用于计算常见等离子体物理参数，并支持 PIC (Particle-in-Cell) 模拟的 CFL 条件验证。

## 特性

- **实时计算**：输入参数后立即更新结果。
- **支持多种单位**：
  - 电子温度：eV 或 K
  - 电子密度：m⁻³ 或 cm⁻³
  - 中性气体压强：Pa（默认）、Torr、mTorr
- **计算参数**：
  - 德拜长度 (λ_D)
  - 等离子体参数 (N_D)
  - 电子惯性长度 (δ_e)
  - 电子/离子等离子体频率 (f_pe, f_pi)
  - 截止频率
  - 电子/离子回旋频率 (f_ce, f_ci)
  - 电子/离子回旋半径 (r_ce, r_ci)
  - 电子热速度 (v_te)
- **PIC 模拟支持**：
  - 输入时间步长 dt 和空间步长 dx
  - 自动验证常见 CFL 条件（不满足时红色警示）：
    - ω_pe × dt < 0.2
    - dx / λ_D < 0.5
    - v_te × dt / dx < 0.5
    - ω_ce × dt < 0.05
- **离子种类**：支持 H、He、Ar（默认）、Kr、Xe
- **磁场支持**：输入磁场强度 B (Tesla)

## 使用方法

1. 将仓库中的 `index.html` 文件下载到本地。
2. 用浏览器打开即可使用。
3. 修改输入参数，计算结果会实时更新。
4. 或直接访问page：https://chen-long-1988.github.io/PlasmaPara/
